hexo博客的本地搭建

一、hexo的环境配置

主要是为下面Hexo的安装运行事先准备一个运行环境，主要包含Node.js和Git两个软件的安装。

​ 1、点击安装Node.js请选择10以上版本和Git.

1.登录网址https://nodejs.org/en/download/；找到板块Download 。
2.根据电脑的类型，选择相应的版本进行下载（那么如何确定自己电脑的类型呢？解决办法：计算机->右键->属性，查找得出本机是64位架构 ）因此，我选择：

​

安装好后，右击空白屏幕，点击Git bash here,输入npm -v

2、点击安装Git,完成后在CMD窗口中输入下面指令:git

二、安装Hexo

1、安装Hexo: 在CMD窗口使用下面的命令:npm install hexo-cli -g

​ 如果不放心，可以输入以下指令来核查：hexo -v

2、创建Hexo博客根目录

​ 因为我要建在D盘：d: 回车,然后md blog（注意，上述语句会在CMD的当前路径下创建一个名为blog的文件夹。该文件夹后面将保存你的各种博客资源及博文，我们把这个目录简称为根目录。建议将目录生成在你常用的位置）

​ 3、初始化Hexo博客：依次执行下面两条命令cd blog ,hexo init。（若实验结果出现“Start blogging with Hexo!”字符串，则表示初始化成功且可以开始写博客了。注意：在初始化Hexo博客之前，必须保证根目录为空,否则无法成功初始化。）

三、博客（本地）预览

​ 1、启动Hexo服务

​ 输入hexo s。

​ 若结果发现有”Hexo is running at http://localhost:4000/字符串，则表示启动成功，Hexo当前 就运行在上述网址。上述指令中的s表示server。

2、博客预览：在浏览器中输入上述网址http://localhost:4000/,如果看到新页面则大功告成。
